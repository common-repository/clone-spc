== Description ==

This plugin allows you to clone Clone SPC theme files from the administrative panel, with the option to add files to the Template Name.
This plugin will be useful when using the advanced hierarchy template themes wordpress.<br>
As we know from the Code (more details - http://codex.wordpress.org/Template_Hierarchy ):<br>
Categories Single entry pages etc - can have its own templates for ID or SLUG <br>
<br>
To once again not to climb on the FTP in the theme folder, you can plug in a copy of the code of the desired file, and assign any new file name.

p.s. Sorry, this text is translated G Translator

== Installation ==

1. Download the plugin
2. Download folder, clone-spc in the directory '/wp-content/plugins/'
2. Activate the plugin
3. Look in the Options section, click on the Clone SPC.
